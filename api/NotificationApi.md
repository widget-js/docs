# NotificationApi

NotificationApi提供了发送通知的能力  

## error

- _**message**_: string  

- _**duration**_: number  持续时间，单位毫秒，默认5000  

## hide

## info

- _**message**_: string  

- _**duration**_: number  持续时间，单位毫秒，默认5000
