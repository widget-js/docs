# WidgetPackageApi

WidgetPackageApi 提供了组件包的升级、获取、安装等功能  

## getEntryUrl

获取组件包入口url信息  

## getIndexUrl

获取组件包的首页url信息  

## getPackage

通过包名获取组件包信息  

## getPackages

获取已经安装的组件包  

## install

安装组件包  

## upgrade

升级组件包  

- _**packageName**_: 组件包名称  

# WidgetPackageApiEvent

|Event|Comment|Payload|
|---|---|---|
|PACKAGE_UPGRADE|组件包升级事件||
|PACKAGE_INSTALLED|组件包安装事件||
